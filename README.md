# SHOUCHIKU 自動見積システム

PDF（車検証＋見積書）をドラッグ＆ドロップするだけで、NEOファイル（自動車板金見積データ）を自動生成するツールです。

## これは何？

- 車検証PDFと損保会社の見積PDFを読み取り
- 部品番号・価格をAddataデータベースと照合
- 自動見積システム（NEO）にそのまま取り込めるファイルを出力

非エンジニア向け。**ダブルクリックで起動**します。

## 使い方（3ステップ）

1. **`起動.bat` をダブルクリック**
   ブラウザが自動で開きます（http://localhost:8501）。
2. **PDFを2枚アップロード**
   車検証PDF と 見積書PDF を画面にドロップ。
3. **「解析開始」→ 数秒待つ → 「ダウンロード」**
   出力された `.neo` ファイルをそのまま自動見積システムへ。

詳しくは `操作マニュアル.md` を参照。

## DB照合を有効にするには

部品番号・価格の自動照合（L1〜L4 マッチ）を有効化するには、
社内 Addata データベース（`C:\Addata` 配下）が必要です。

サイドバーの「Addataで部品照合する」をオンにしてください。
DBが無い環境では照合はスキップされ、手入力ベースで動作します。

## 困ったとき

| 症状 | 対処 |
|------|------|
| ブラウザが開かない | 手動で http://localhost:8501 を開く |
| 「ポート使用中」エラー | 既に起動済み。ブラウザだけ開けばOK |
| OCRが失敗する | `.env` の `GEMINI_API_KEY` を確認 |
| 何もうまく行かない | コンソール（黒い画面）を閉じてから再度 `起動.bat` |

困ったら 操作マニュアル.md / デプロイ手順書.md を参照。
