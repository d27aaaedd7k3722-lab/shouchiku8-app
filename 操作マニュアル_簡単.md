# SHOUCHIKU8 NEO 簡単マニュアル（v11.0）

## このアプリで何ができるか

工場が作った **見積 PDF**（紙でもFAXでも）を、コグニセブンで開ける **.neo ファイル** に自動変換します。

- 1件あたり **30秒〜2分** で完了
- 部品名・数量・金額をすべて自動で読み取り
- PDF の合計金額に **完全一致** するように調整
- 車検証 PDF があれば、車両情報も自動反映

## 使い方（3ステップ）

### ① ブラウザで本番URLを開く

```
https://shouchiku-shouchiku8-neo.hf.space/
```

「簡単モード」がデフォルトで開きます。

### ② PDF を投げ込む

画面に **3つの枠** があります：

| 枠 | 入れるもの | 必須？ |
|---|---|---|
| ① 見積 PDF | 工場からもらった見積（コグニ／他社問わず） | **必須** |
| ② 車検証 PDF | 車検証の PDF | 任意（精度↑） |
| ③ 原本 NEO | 以前作成した .neo ファイル | 任意（保険情報引継ぎ） |

各枠に **PDF をマウスでドラッグ&ドロップ** するだけです。

### ③ 「📥 NEO ファイルを作成」ボタンを押す

進捗バーが進み、30秒〜2分で完了します。

## 完成画面の見方

完了すると、結果カードが表示されます：

- ✅ **完成しました** ← 成功
- 明細件数 / PDF 総額 / 処理モード
- **［📥 .neo ダウンロード］** ← クリックで .neo ファイル保存
- **［📋 明細 CSV］** ← Excel で確認したい時用

ダウンロードした `.neo` を **コグニセブンで開く** だけで、見積データが取込まれます。

## 困った時

### エラーが出た場合

赤い枠で「**何が起きたか / 次にどうするか**」が表示されます。書かれた手順に従って下さい。

例：
- 「⏱️ AI処理がタイムアウトしました」→ もう一度ボタンを押す
- 「📊 AI利用枠を使い切りました」→ 明日もう一度（または管理者に「pro モードに切替えて」と相談）
- 「📄 PDF ファイルが壊れています」→ PDF を保存しなおして再投入

### それでもダメな時

画面のスクリーンショットを撮って、亮平さんに送って下さい。

### サイドバーの「簡単モード」を OFF にすると

詳細モード（旧 UI）に切り替わり、ADDATA フォルダ指定や精度モード切替などの細かい設定ができます。社内の管理者向けです。

## よくある質問

**Q. 「処理モード」の A / B / C って何？**
- **A**：ベタ打ち（PDF の値をそのまま .neo に書く）
- **B**：コグニ完全複製（コグニ作成見積を自動で.neoに）
- **C**：あいまい検索（他社見積の部品をADDATAで照合）

どのモードでも **PDF と完全一致した金額** で .neo が作られます。

**Q. PDF と NEO で金額が違うように見える**
コグニセブン取込時に **税が再計算される設計** です。.neo 内では税抜で保存し、コグニセブンが10%を加えて税込表示します。**最終的に表示される金額は PDF と完全一致します**。

**Q. ADDATA って何？**
コグニセブンの部品データベース（A〜Zフォルダの .DB ファイル）。社内 PC の `C:\Addata` または OneDrive に置いてあります。本番URL（HuggingFace Space）では使えませんが、**ベタ打ちモード（A）で完全動作** します。

**Q. パスワードは？**
要りません。社内ブラウザでURLを開いたら即使えます。
